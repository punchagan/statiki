Statiki is simple way to create and/or manage a website or blog.

Statiki leverages free and open services and software like GitHub, TravisCI and Nikola.

How do I get started?
Create an account on GitHub

If you already have one, login. Statiki will uses GitHub OAuth to authenticate.

Create an account on TravisCI

TravisCI also uses GitHub OAuth. Once you create your account, wait for Travis to sync your repositories from GitHub

Create an account on Statiki

Statiki uses GitHub OAuth. Statiki only uses your authentication to manage the repositories you specify. Once Statiki sets up your repository, with the required files, and the hook on TravisCI, you can even revoke access to Statiki.

Choose a URL for your site and hit Go!

Currently, only urls of the type http://<your-username>.github.io/<choosen-name> are supported by Statiki. In future Statiki, will be able to let you use just http://<your-username>.github.io/> RSS feeds/Galleries may not work correctly, due to an issue in Nikola.

Profit!

Your site should be ready in a few minutes.

You'll get a notifiation email from GitHub! (Actual time depends on the availability of TravisCI and how long GitHub takes to deploy your pages.) You can edit your site either using GitHub's web editor, or edit locally using your favorite editor and pushing changes to your git repository!
